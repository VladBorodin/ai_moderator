# AI Moderator

AI Moderator - это Python/FastAPI приложение-прокладка для контекстной модерации сообщений.

Приложение получает JSON с коротким фрагментом переписки, анализирует последнее сообщение и возвращает результат модерации.

На текущем этапе проект содержит:
- FastAPI API;
- PostgreSQL через Docker Compose;
- SQLAlchemy модели;
- автоматическое создание таблиц при запуске приложения;
- базовый endpoint `/moderation/check`;
- настройки через `.env`;
- подготовку к будущему подключению локального или внешнего ИИ.

## Текущий статус

ИИ-провайдер пока не подключается в финальном виде.  
Сначала поднимается каркас приложения, БД, DTO, модели и API.

## Требования

Для холодного запуска нужны:

- Python 3.12+
- Git
- Docker Desktop
- PowerShell или CMD
- доступ к интернету для установки Python-зависимостей
